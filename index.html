PRISK HOSPITAL ELECTRONIC GATE PASS

PROJECT.

An electronic gate pass for clients visiting Prisk hospital.


PROBLEM STATEMENT

Having visited a few hospitals during this Covid 19 pandemic, I realized new measures have been put in place with regard to accessing the hospital for whatever reason. Acquiring a gatepass is done manually which is not only time consuming but tiresome given the large number of people visiting. I figured coming up with an electronic gate pass system would be efficient, convenient and time saving.
This project is still at the ideation stage, I am open to collaborating with interested parties.
